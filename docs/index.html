langchain-middleware-stack

Declarative ordering for LangChain agent middleware — DAG constraints, stable resolution, zero runtime dependencies.

Install

pip install langchain-middleware-stack

Primitives

  • slug — stable unique id per middleware
  • after / before — ordering constraints (resolved to outer→inner list)
  • MiddlewareStack — .add(...) then .resolve()
  • wires — optional attribute injection from upstream instances at resolve time
